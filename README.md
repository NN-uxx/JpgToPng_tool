# ツールの名前
JpgToPng_tool ツール

## 簡単な説明
Jpgの画像ファイルをpngの画像ファイルに変換します。
逆、pngをJpgに変換する事はできません。

## 必要要件
Windows11

## 使用ライブラリー
PIL

## 使い方
1・適当なフォルダに変換したい画像ファイル（Jpg）を入れてください。（面倒なら同梱のinputフォルダに入れてください）<br>
2・JpgToPng.pyをクリックしてください。<br>
3・1で選択したフォルダの下にoutputフォルダが作成され、変換されたpngが収められます。<br>　　

## インストール
解凍するだけ。アンインストールはフォルダごと消してください
